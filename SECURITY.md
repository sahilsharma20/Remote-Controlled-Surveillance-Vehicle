# Security and Safety Policy

## Reporting

Please report software vulnerabilities privately to the repository maintainer rather than opening a public issue with exploit details.

## Physical safety

This repository controls real hardware. Test with the servo unloaded, use conservative angle limits, provide a manual power disconnect and keep people clear of moving mechanisms.

The prototype media includes a non-functional demonstration prop. This project is for educational computer-vision and robotics work and is not intended for weaponization, targeting or harmful surveillance.
