# GitHub Repository Setup

Use the following repository metadata to make the project easier to discover.

## Suggested description

Computer-vision robotics prototype using MediaPipe Pose, OpenCV, Arduino and serial servo control for real-time human tracking.

## Suggested topics

`computer-vision`, `robotics`, `mediapipe`, `opencv`, `arduino`, `pose-estimation`, `human-tracking`, `pyserial`, `embedded-systems`, `python`

## Recommended repository settings

- Enable Issues and Discussions.
- Keep the default branch named `main`.
- Add branch protection after CI passes reliably.
- Require the `quality` workflow before merging pull requests.
- Create a first release such as `v1.0.0` only after testing the packaged code with the actual hardware.
- Pin the repository on the GitHub profile.

## Suggested first professional commit

```bash
git add .
git commit -m "refactor: professionalize vision-guided rover project"
git push origin main
```
